Floating Chat Widget

This example shows the floating chat bubble in the bottom right of the screen. Click it to open the chat window.

The floating widget is ideal when you want to provide chat functionality without taking up space on your page.

Inline Chat Widget

This example shows the chat widget embedded directly in the page using an iframe.

The inline widget is ideal when you want to integrate the chat experience directly within your page layout.

Code Examples

Floating Widget Code

<!-- EzAIBotz Chat Widget -->
<script 
  src="/widget.js" 
  data-chatbot-id="YOUR_CHATBOT_ID"
  data-primary-color="#4F46E5" 
  data-api-url="/api/chat"
  data-welcome-message="Hello! How can I help you today?">
</script>

Inline Widget Code

<iframe
  src="/embed/YOUR_CHATBOT_ID"
  width="400"
  height="600"
  frameBorder="0"
  title="Chat with EzAIBotz"
  style="border: 1px solid #e5e7eb; border-radius: 0.5rem;"
></iframe>
